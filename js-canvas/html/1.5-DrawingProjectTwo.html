기본 전제

const canvas = document.querySelector("canvas");
const ctx = canvas.getContext("2d");

canvas.width = 800;
canvas.height = 800;

<!-- 얼굴, 몸 만들기 -->
ctx.fillRect(200, 200, 15, 100); 좌측 팔
ctx.fillRect(400, 200, 15, 100); 우측 팔
ctx.fillRect(250, 200, 115, 300); 몸통
ctx.arc(305, 130, 50, 0, 2 * Math.PI);
ctx.fill();

arc ⇢ x, y, radius, startAngle, endAngle, …

endAngle 원을 끝내는 angle

<!-- 눈 만들기 -->
ctx.beginPath();
ctx.fillStyle = "white";
ctx.arc(285, 120, 10, 0, 2 * Math.PI);
ctx.arc(325, 120, 10, 0, 2 * Math.PI);
ctx.fill();

#02에서 배웠던 beginPath()

beginPath를 이용해 새로운 경로를 만들고 fillStyle 컬러 변경



오른쪽부터 시계 방향대로

0 * Math.PI

0.5 * Math.PI

1 * Math.PI

1.5 * Math.PI

2 * Math.PI

endAngle 값

우측 끝이 0, 하단이 0.5, 좌측이 1, 상단이 1,5, 우측 끝(360도) 2

0 * Math.PI를 fill을 한다면 아무것도 채워지지 않는다.

Math.PI의 장점 *PI값을 이용하면 내가 정확한 원을 만들지 않아도 자동으로 만들어주기에 간편함
