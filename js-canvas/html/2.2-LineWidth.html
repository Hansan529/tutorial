-html-
<input type="number" />
<input id="line-width" type="range" min="1" max="50" vlaue="2" step="0.1"/>


-js-
const lineWidth = document.getElementById("line-width");
const fontsize = document.querySelector("input");
ctx.lineWidth = lineWidth.value;


-Function-
function range(event) {
fontsize.value = event.target.value;
ctx.lineWidth = event.target.value;
}

function inputNumber(event) {
lineWidth.value = fontsize.value;
ctx.lineWidth = event.target.value;
}


-EventListner-
lineWidth.addEventListener("change", range);
fontsize.addEventListener("change", inputNumber);

input tag style:인스타일, type:종류 min,max:최소 최대값, value:기본값, step:단위
타입이 range인 input을 생성하여 값을 선택 할 수 있도록 했다.
변경되는 값에 대해 event를 넣었다. change되면 range 함수를 실행함
event 값을 받아와서 lineWidth에 적용

inputBox도 동일하게 진행함. 이 과정중에 range와 값을 동기화하기 위해 서로를 복제함
