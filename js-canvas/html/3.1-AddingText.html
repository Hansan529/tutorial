text input을 만들기, user가 텍스트를 입력할 수 있고, canvas를 더블 클릭하면 해당 text를 추가한다. 

      const textInput = document.getElementById("text");

      ctx.lineCap = "round";

      function onDoubleClick(event) {
        ctx.save();
        const text = textInput.value;
        ctx.lineWidth = 1;
        ctx.strokeText(text, event.offsetX, event.offsetY);
      }
      

      canvas.addEventListener("dblclick", onDoubleClick);

기존과는 다르게 eventListener를 추가하지 않는다. doubleClick 이벤트는 mousedown, mouseup이 빠르게 발생하면 발동되는 이벤트다.

function onDoubleClick의 ctx.strokeText, ctx.fillText 둘 중 원하는걸 사용하면 된다.
Text의 outline을 두껍게하면 보이지 않으니 lineWidth를 1로 변경했다. 그랬더니 선들의 두께도 줄어들었다.
이 문제를 해결하기 위해서 save 함수를 사용한다. save 함수는 현재 상태, 색상, 스타일 등 모든 것을 저장한다.

font는 size와 font-family를 지정할 수 있다.
lineCap은 선들의 타입을 설정한다. butt, round, square로 설정할 수 있다.
