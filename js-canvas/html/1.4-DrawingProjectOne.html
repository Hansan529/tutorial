기본 전제

const canvas = document.querySelector("canvas");
const ctx = canvas.getContext("2d");

canvas.width = 800;
canvas.height = 800;

ctx.fillRect(200, 200, 50, 200);
ctx.fillRect(400, 200, 50, 200);
ctx.lineWidth = 2;
ctx.strokeRect(300, 300, 50, 100);
ctx.fillRect(200, 400, 250, 20);
ctx.fillRect(200, 200, 200, 20);
ctx.moveTo(200, 200);
ctx.lineTo(325, 50);
ctx.lineTo(450, 200);
ctx.fill();

200,200 위치에서 너비 50, 높이 200 400,200 위치에서 너비50, 높이 200를 채우는 벽을 만든다.

ctx.lineWidth = 2 ⇢ 이후 사용할 strokeRect의 두께를 변경하기 위해 lineWidth를 2px로 설정한다.
두께 2px의 x 300, y 300, 가로 50, 세로 100의 테두리를 만듦


x 200, y 400, w 250, h 20 바닥,
x 200, y 200, w 200, h 20 천장 만들기

moveTo x 200, y 200 선 이동
moveTo 위치에서 x 325, y 50 그리기 ↗, 325, 50에서 450, 200으로 그리기 ↘
