html
      <input type="file" accept="image/*" id="file" />

javascript
      const fileInput = document.getElementById("file");
      
      function onFileChange(event) {
        const file = event.target.files[0];
        const url = URL.createObjectURL(file);
        const image = new Image()
        image.src = url;
        image.onload = function () {
          ctx.drawImage(image, 0, 0, CANVAS_WIDTH, CANVAS_HEIGHT);
          fileInput.value = null;
        };
      }

input type="file" 해당 코드만 입력하면 사진, 영상 모두 선택이 가능하다. accept를 사용하면 내가 원하는 타입만을 불러 올 수 있다.
또한 해당 파일의 포멧 종류도 설정 가능하다.


브라우저는 내 파일을 읽을 수 없기에 저장 할 수 있도록 input을 만들고 업로드하면 브라우저에 저장된다.

const image = new Image() === document.createElement("img") 동일 JS:HTML image.src === <img src=""></img> 동일

canvas.onclick = onCanvasClick;

동일한 결과를 나타낸다. 다만 addEventListener의 경우 같은 event 안에 많은 event listener를 추가,삭제를 할 수 있다.

drawImage : 이미지, 위치(가로, 세로), 크기(가로,세로)

fileInput.value를 null 하는 이유는 단지 '선택한 파일 없음'이라고 나타나게 하기 위함임
