이미지를 저장할 수 있는 버튼을 만들기

const saveBtn = document.getElementById("save");

function onSaveClick() {
  const url = canvas.toDataURL();
  const a = document.createElement("a");
  a.href = url;
  a.download = "myDrawing.png";
  a.click();
}


saveBtn.addEventListener("click", onSaveClick);

toDataURL base64로 인코딩된다. image를 클릭하면 해당 경로로 이동하게 만듬
html a 태그 안에 있는 download라는 속성을 이용한다. a라는 가상 태그를 만들고 href 속성에 url를 기입하고
download 될 이름,포맷을 지정한다. 그리고 click 시키면 끝
