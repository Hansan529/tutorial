주축(main axis)과 교차축(cross axis)이 있음. 주축은 수평이며 교차축은 수직이다.

justify-content는 주축에 해당한다. align-items는 교차축에 해당한다.

기본적으로 주축은 수평, 교차축은 수직으로 설정되어있고 이를 변경 할 수 있다.

body { display:flex; justify-content: center; align-items: center; height: 20vh; }

…

같은 스타일에 height 값을 주면 (vh는 viewport height이다) body의 center로 div들이 이동을 하게 된다.

align-items은 justify-content에 비해 사용 할 수 있는 옵션이 적다.

예를 들어 stretch 값을 준다면

div { height: 50px; width: 50px; background-color: orange;

…

200x200 도형이 orange 색상의 배경을 갖고 있는데, 뷰포트가 사이즈가 0부터 최대 700까지 위치할 경우 div의 배경색을 blueviolet으로 변경한다.

@media screen and (min-width: 600px) and (max-width:800px) {
  div {
    background-color: navy;
  }
}

뷰포트가 최소 600부터 최대 800 사이에 있을 경우에 div의 색상을 변경한다.

@media screen and (min-width: 600px) and (max-width:1200px) and (orientation: landscape) {
  div {
    background-color: aqua;
  }
}

최소 600 - 최대 1200 사이에 가로 모드일 경우 aqua 색상을 적용한다.

@media screen and (orientation: landscape){
div + span {
display: none;
}
div {
background-color: black;
}

display:inline;
color: white;
}
}
