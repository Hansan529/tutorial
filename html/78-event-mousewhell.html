mousewheel - 크롬, 사파리, 오페라, 익스

DOMMouseScroll - 파이어폭스

e.originalEvent의 detail 값이 있다면 파이어폭스 브라우저이며 '0' 일 경우 다른 브라우저

휠을 아래 방향으로 향할 때, detail 값이 파이어폭스: '+', 다른 브라우저: '-' 값이 나온다 

if (event.detail) {
      // 파이어폭스
      delta = -event.detail;
    } else {
      // 다른 브라우저
      delta = event.wheelDelta;
    }
