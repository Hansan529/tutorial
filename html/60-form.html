form

focusin: 포커스가 되면 실행함

focusout: 포커스에서 벗어나면 실행함

focus, blur: toggle과 같이 포커스가 되면, 벗어나면 실행함

        "focus": function () {
          $(this).addClass("bg2");
        },
        "blur": function () {
          $(this).removeClass("bg2");
        }

select: value 값이 드래그가 되면 실행함

change: 변경되면 실행함

.val(): 변경된 값을 얻어냄. .prop("selectedIndex"): 몇 번째인지 순서를 알려줌
